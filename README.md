# 概要
beatmania IIDXのスコアを管理できるWebアプリケーション

# 使い方
公式サイトからダウンロードしたCSVファイルをアップロードすることで、スコア（記録）の一覧表示を行えます。  
メニューからレベル別・クリアタイプ別・クリアランク別・バージョン別に表示できます。  
管理者アカウントは、ユーザの登録情報の編集とアカウント削除が可能です。
